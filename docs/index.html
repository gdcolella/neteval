How it works

1

Download & run

Download the binary for your OS. Double-click it. A dashboard opens in your browser.

2

Discover machines

Enter your Active Directory credentials. NetEval finds every Windows computer on your domain and deploys a lightweight agent to each one.

3

Run tests

Click "Run Mesh Test." NetEval measures upload and download speed between every pair of machines, plus each machine's internet speed.

4

See the heatmap

Results appear in a color-coded matrix. Green means fast, red means slow. Instantly see which links are the bottleneck.

Features

Single binary

One file, zero dependencies. No runtime, no installer. Works on Windows, macOS, and Linux.

Full mesh testing

Tests every pair of machines on your network using round-robin scheduling so tests don't interfere with each other.

Active Directory integration

Discovers machines via LDAP and deploys agents over SMB. No manual setup on each computer.

Real-time dashboard

Web-based UI with live agent status, a speed heatmap, and an activity log. Runs in any browser.

Internet speed testing

Each agent tests its own upload and download to the internet so you can compare LAN vs. WAN performance.

No port forwarding

Agents connect outbound to the coordinator. Works through NAT and firewalls on your LAN out of the box.
